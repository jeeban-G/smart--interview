# 面试界面专注模式设计

## 背景

当前面试界面太狭小，信息拥挤，用户体验不佳。需要改造成更宽广、沉浸的界面。

## 设计目标

1. **宽银幕布局** - 最大化利用屏幕宽度和高度
2. **聊天软件风格** - Avatar 在消息旁边，微信/WhatsApp 风格
3. **情绪状态** - 两个 AI Agent 头像显示情绪（思考中、开心、难过等）
4. **简洁底部** - 控制按钮不抢戏

## 功能细节

### Avatar 设计
- 大小：32x32px（小而精致）
- 位置：消息气泡外侧
- 候选人头像在左侧，面试官头像在右侧
- 情绪状态以 emoji 叠加在头像上

### 情绪状态
| 状态 | Emoji | 触发场景 |
|------|-------|----------|
| 思考中 | 🤔 | 等待回复、分析问题时 |
| 开心 | 😊 | 回答满意、被采纳 |
| 难过 | 😟 | 回答被拒绝、遇到困难 |
| 中立 | 😐 | 正常对话中 |
| 认真 | 🤨 | 听到重要信息 |

### 消息布局
- 候选人消息：头像在左，对齐左侧
- 面试官消息：头像在右，对齐右侧
- 消息气泡：圆角矩形，背景色区分
- 消息间距紧凑但有呼吸感

### 顶部栏
- 简化：只保留职位名称和面试类型标签
- 去掉多余的 padding

## 实现要点

1. 修改 `MessageListManga.tsx` 或创建新组件
2. 传入当前情绪状态给消息列表
3. 根据 sender_type 决定 avatar 位置
4. 情绪状态通过 props 传入

## 组件结构

```
Interview.tsx
├── Header (简化版)
├── MessageListManga
│   ├── MessageItem (avatar + bubble)
│   └── 每个消息显示对应agent的avatar和情绪
└── ControlBar (底部按钮)
```
